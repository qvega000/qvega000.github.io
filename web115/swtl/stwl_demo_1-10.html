SWTL Demonstrate Chapters 1-10

Chapter 1: Alerts
Alerts: A box that pops with a message. alert("Hey Quentin"); Displays: Hey Quentin
Chapter 2: Variables for Strings
Variables: A word or number that holds a value var name = "Quentin"; name = "Quentin";
Chapter 3: Variables for numbers
Variables: A word or number that holds a value var weight = 150; weight = 150;
Chapter 4: VariableName Legal and Illegal
Variables names can not be a keyword. Variable names can't have spaces Variable names can have letters, numbers, $, and underscores. Legal: var myName; Illegal: var class;
Chapter 5: Math expressions: Familiar operators
Arithmetic operators: +, -, *, /, % var num = 1 + 5 or alert(1 + 5) both display 6
Chapter 6: Math expressions: Unfamiliar operators
num++, num--, ++num, --num var num = 5; var newNum = num++; Displays: 6
Chapter 7: Math expressions: Eliminating ambiguity
The order of operations var newNum = 1 + 5 * 6; 5 * 6 then add 1 newNum = 31;
Chapter 8: Concatenting text strings
Using the + sign to combine a personalized message to the user. var userName = "Quentin" var message = "Hey " alert(message + userName); Displays: Hey Quentin
Chapter 9: Prompts
Prompts: Dialog box that ask the user for information and they provide area for the users answer. var favFood = prompt("What is your favorite food?, Spagetti");
Chapter 10: if statements
If statements: Test for true or false answers. var greeting = prompt("Are you new to this website?); if (greeting === "Yes"){ alert("Welcome!");} else { alert("Welcome back!!);}
